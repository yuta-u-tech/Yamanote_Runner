# 要件メモ

## v0.1 MVP

山手線ランナーの初期版では、HealthKit から取得した歩行・ランニング距離を山手線上の進捗に変換し、現在位置、次の駅までの距離、周回数を表示します。

## 初期スコープ

- HealthKit 連携
- 今日の歩行・ランニング距離の取得
- 前回同期との差分加算
- 山手線ステージ
- スタート駅選択
- 現在位置表示
- 次の駅までの距離表示
- 周回数表示
- 最小限のバッジ・称号

## 初期スコープ外

- 複数路線
- 乗り換え
- カスタムルート
- SNS 共有
- ランキング
- 広告
- アプリ内課金

## Vol.2 候補

Vol.2では、v0.1の山手線進捗を壊さずに散歩マップ体験を追加する。HealthKit距離同期、履歴、バッジは既存ロジックを継続利用し、地図機能は独立した機能境界として追加する。

初期の土台方針は [vol2-foundation.md](vol2-foundation.md) にまとめる。
